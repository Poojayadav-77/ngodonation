# NGO Donation System

A MERN-based donation platform for NGOs enabling secure donor management.

## Tech Stack
Node.js | Express.js | MongoDB | JWT | HTML | CSS | JavaScript

## Features
- JWT-based user authentication
- 500+ donor records management
- Full CRUD operations for donors
- Responsive UI with fast query response

## Setup
1. Clone the repo
2. Run `npm install`
3. Add `.env` with your MongoDB URI
4. Run `node server.js`
